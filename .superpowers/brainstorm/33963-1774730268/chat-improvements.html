Chat Experience: Current Issues & Direction

Let's pick a design direction for the chat overhaul

Current Problems

VISUAL
  • Send button is gray (#6d6867) — no personality
  • No distinction between user/assistant messages
  • 3 stacked bars at bottom (session + model + input)
  • Code blocks at 10px — too small for mobile
  • Markdown body at 12px — needs to be larger
UX
  • "Thinking" indicator above the message list
  • Tool calls just show "tool_name..." as text
  • Model picker is a full bar, always visible
  • No streaming animation/progress
  • Messages reversed with .reverse() — scroll bugs
